# BandToy Product Brief

BandToy is a "smart figurine that can form a band".

Each character is a small music companion with a personality, an instrument, and
a limited songbook. When several characters are placed together, they can
discover each other and join the same performance.

BandToy should feel closer to:

- a music box
- a tiny pet
- a magical object from a quiet animated film
- a friend that likes playing music with other friends

BandToy should not feel like:

- an AI assistant
- a smart speaker
- a robot demo
- a technical music algorithm

## PoC Question

The only question for the first prototype:

> When one character starts playing and another character automatically joins,
> is the scene emotionally interesting enough?

If yes, continue. If no, rethink the product.

## Experience Priority

1. Emotional experience.
2. Sense of life.
3. Expandability.
4. Technical elegance.

Technical choices should serve the illusion that these little characters like
playing music together.

## First Scene

Character A starts playing the melody.

Character B notices.

Character B wakes up with light.

Character B waits one or two bars.

Character B joins with harmony.

The important feeling is not "sync". The important feeling is "it listened, it
thought, and it chose to join".
